# Mobile Flashcards

This is a mobile app to create flash cards.

User can categorize flash cards into collections.

The app is built with React Native; it is available on both iOS and Android.


## Requirements
Since this app is not published on the App Store, there are a few requirements to launch.
If you already have yarn or npm installed on your machine, and you have Expo or Genymotion, skip over to the launch settings. This project recommends you make use of Expo to save time.

1. Download NodeJS here:
[https://nodejs.org/en/](https://nodejs.org/en/)

2. Download Expo on your machine:
After NodeJS has been installed, go into your Command Line and type the following:
```
npm install expo-cli --global
```

3. Expo
You can run it either in the simulator or on a physical device.
To take it out for a spin in a mobile device, you will need to download the Expo app from either the iOS App Store or the Android Play Store.
iOS
[https://itunes.apple.com/app/apple-store/id982107779](https://itunes.apple.com/app/apple-store/id982107779)
Android
[https://play.google.com/store/apps/details?id=host.exp.exponent&referrer=www](https://play.google.com/store/apps/details?id=host.exp.exponent&referrer=www)

You might want to check out [Genymotion](https://www.genymotion.com/) for an Android simulator to run on your computer.


## Launch It
1. Download this project
You can get a copy of this project on your machine by clicking the green `Clone or download` button on this page: [https://github.com/call900913/react-rchristy-project3](https://github.com/call900913/react-rchristy-project3).

Alternatively,  you can enter your machine's Command Line and type the following code in the folder of your choosing:
```
git clone https://github.com/call900913/react-rchristy-project3.git
```

2. Navigate into the project root directory
Type `yarn install` or `npm install`
Next, if you are using Expo, type `expo start`.
If you have a device on the same network on your computer, you can launch the app on the device.
